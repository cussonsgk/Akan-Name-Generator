# Akan Name Generator

## Author
Lucy Kamau

## Description
A web plication that calculates the day of the week the user was born and gives them an Akan name based on the day of the week they were born in and their gender.

## Setup/Usage Instructions
Clone or download this repository
Open the html file in a browser
Enter your birth date and select your gender
Click on the "Get your Akan Name" button
And wait a second for your Akan name to be generated

## Languages Used
HTML, CSS and JavaScript
